## Overview

This is a pokedex app showcasing the use of Next.js, Shadcn UI, Clerk, and React Query. It is a simple pokedex app that allows you to search for a pokemon by name and view its details, abilities, and moves. Loop up top 151 pokemon, collect favorite pokemon, and view your collection using React Query.

**Scope:** A pokedex app showcasing the use of Next.js, Shadcn UI, Clerk, and React Query. Where I used the free pokemon API to fetch pokemon data and store it in cache for faster loading times using Tanstack - React Query.

**Tech stack:** Next.js, Shadcn UI, Clerk, React Query, GraphQL, TypeScript, Tailwind CSS, JavaScript, Git

- Built a pokedex app using Next.js, Shadcn UI, Clerk, and React Query
- Integrated the PokeAPI to fetch pokemon data
- Used Shadcn UI for the UI components
- Used Clerk for authentication
- Used React Query for data fetching
- Used GraphQL to fetch pokemon data
- Used TypeScript for type safety
- Used Tailwind CSS for styling
- Used JavaScript for the frontend
- Used Git for version control

## Key Features

- Search for a pokemon by name
- View a pokemon's details, abilities, and moves
- Loop up top all available pokemon in the Pokemon API
- Collect favorite pokemon and view your collection
- Login and logout using Clerk
- Responsive design
- Persist favorite pokemon in local storage

## Installation

```bash
git clone https://github.com/aaronm-git/pokedex-next-clerk-react-query.git
cd pokedex-next-clerk-react-query
npm install
npm run dev
```

### Environment Variables

```bash
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
NEXT_PUBLIC_CLERK_SIGN_IN_FORCE_REDIRECT_URL=/app/dashboard
NEXT_PUBLIC_CLERK_SIGN_UP_FORCE_REDIRECT_URL=/app/dashboard
```
